# Creators Tank with Convex 🔥

## What You Got

✅ **100% FREE** - No credit card, no surprise bills  
✅ **Real-time magic** - All 300 phones update instantly when you click "Next Team"  
✅ **Beautiful UI** - Dark theme with smooth animations  
✅ **Super easy setup** - Just `npx convex dev` and you're done!

## Quick Start

```bash
# 1. Install
npm install

# 2. Set up Convex (opens browser, sign up with GitHub/Google)
npx convex dev

# 3. In a NEW terminal, start the app
npm run dev

# 4. Go to http://localhost:3000/setup and add your teams!
```

## Event Day Flow

1. **Admin** opens `your-app.vercel.app/admin`
2. **Students** open `your-app.vercel.app` on their phones
3. **When reel starts**, admin clicks "Next Team"  
4. **BOOM!** All 300 phones update instantly! 🚀
5. **Students** tap the like button
6. **Everyone** sees likes increase in real-time
